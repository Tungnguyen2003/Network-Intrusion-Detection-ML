Project Introduction: Giới thiệu về hệ thống phát hiện xâm nhập (NIDS) sử dụng tập dữ liệu CIC-IDS2017.

Installation Guide: Liệt kê các thư viện cần thiết như pandas, numpy, scikit-learn, seaborn, matplotlib.

Usage Instructions: Hướng dẫn cách chạy mã nguồn (ví dụ: chạy tệp .ipynb trên Jupyter Notebook).

Summary of Results: Phần này bạn sẽ cập nhật sau khi hoàn thành việc so sánh các mô hình ở bước 2.5.
